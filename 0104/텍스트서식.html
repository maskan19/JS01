블럭 요소 : 한줄 전체를 차지, 크기를 지정할 수 있다.
	인라인 요소 : 지정한 영역을 차지, 크기를 지정할 수 없다.
	가로 길이 지정 : width
	세로 길이 지정 : height
	
	p태그 : 하나의 단락을 지정
	span : 단락 안에서 일부분의 문자를 지정
	hr : 구분선. 구분선 안에 내용이 없을 때는 닫지 않는다.
	&nbsp; : 공백<pre>외에서 사용
	&lt; <
	&gt; >
	& &
